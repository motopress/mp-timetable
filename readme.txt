Create an online schedule for gym classes, conferences, learning lessons etc

== Description ==

Timetable plugin allows to create an online schedule for gym classes, conferences, learning lessons etc. It goes with shortcode generator letting easily add a timetable to the site page and upcoming events widget.

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
